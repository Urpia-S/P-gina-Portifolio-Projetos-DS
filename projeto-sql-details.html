Olist E-commerce Analytics - SQL + Python

Projeto de análise de e-commerce com a base pública da Olist.

Visão Geral do Projeto

Análise exploratória de dados de e-commerce utilizando SQL e Python. Este projeto é focdo especilamente na manipulação das bases relacionais através do SQL e na extração de métricas de performance e comportamento do consumidor.

Estruturação e Etapas do Projeto

Notebook (Pipeline) Conteúdo e Objetivos
01_preparacao_base_colab.ipynb Upload/download de CSVs, carga em SQLite staging, criação de core e views.
02_temporal_receita_colab.ipynb Análise de pedidos por tempo, evolução de receita, frete, segmentação por estados e categorias.
03_entregas_satisfacao_colab.ipynb Monitoramento de prazos, atrasos, reviews de clientes e impacto da logística na satisfação.
04_rfm_retencao_colab.ipynb Segmentação RFM (Recência, Frequência, Valor Monetário), análise de recompra e coortes de retenção.
05_afinidade_vendedores_colab.ipynb Análise de categorias compradas juntas, performance de vendedores e resumo executivo.
06_playground_consultas_sql_colab.ipynb Ambiente para testes de novas consultas SQL complexas e validação de métricas.
